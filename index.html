    Shell from Scratch

    Developed a fully functional custom shell in C++ using system calls. The shell supports a wide range of Unix-like commands and built-in features, including: Command execution: Supports semi-colon separated commands and background processes with the & operator. Input/Output redirection: Efficiently handles file redirection for reading and writing. Pipes: Allows chaining of commands, redirecting output from one to another using pipelines. Built-in commands: Includes custom implementations of commands like cd, pwd, echo, ls, and more, developed from scratch. Process handling: Provides functionality for foreground and background process management. Dynamic prompt: Displays the current user, system name, and directory in the prompt. Search feature: Recursively searches files and directories under the current path. This project showcases my expertise in system programming, process management, and C++ development.
